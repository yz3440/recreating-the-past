## 1. MIT Press Colophon (Logo)

|                  |                                                                                                                                                                                                                                                                                               |
| ---------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Work**         | MIT Press Colophon                                                                                                                                                                                                                                                                            |
| **Year**         | 1962–1965                                                                                                                                                                                                                                                                                     |
| **Medium**       | Logo / Identity Design                                                                                                                                                                                                                                                                        |
| **Context**      | Seven vertical white bars against black, abstracting the lowercase letters "mitp" while evoking book spines on a shelf. Paul Rand was originally offered the commission but recommended Cooper. Acquired by MoMA's permanent collection in 2023. Still in use on every MIT Press publication. |
| **Collaborator** | Design assistant Lauri Rosser                                                                                                                                                                                                                                                                 |

## 2. _The Bauhaus_ — Book Design

|                  |                                                                                                                                                                                                                                                                                                                                        |
| ---------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Work**         | _Bauhaus: Weimar, Dessau, Berlin, Chicago_ by Hans M. Wingler                                                                                                                                                                                                                                                                          |
| **Year**         | 1969                                                                                                                                                                                                                                                                                                                                   |
| **Medium**       | Book Design (MIT Press)                                                                                                                                                                                                                                                                                                                |
| **Context**      | Massive encyclopedic redesign of a German original for the 50th anniversary of the Bauhaus. Set in the then-new Helvetica typeface on a three-column grid. 653 pp., 24 color plates, 1,000+ b/w images. Dominated Cooper's work for nearly two years. Cooper also created a film rendition — her attempt at "turning time into space." |
| **Collaborator** | —                                                                                                                                                                                                                                                                                                                                      |

## 3. _Learning from Las Vegas_ — First Edition Book Design

|                  |                                                                                                                                                                                                                                                                                                                                     |
| ---------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Work**         | _Learning from Las Vegas_ by Robert Venturi, Denise Scott Brown & Steven Izenour                                                                                                                                                                                                                                                    |
| **Year**         | 1972 (first edition)                                                                                                                                                                                                                                                                                                                |
| **Medium**       | Book Design (MIT Press)                                                                                                                                                                                                                                                                                                             |
| **Context**      | The ground-breaking manifesto of postmodernist design. Cooper envisioned it as a monumental (14.5×11 in.) cinematic coffee table book to provide an immersive reading experience akin to driving the Strip. Heavily criticized by the authors; now a cult classic. Very different from the more restrained revised edition of 1977. |
| **Collaborator** | —                                                                                                                                                                                                                                                                                                                                   |

## 4. _File Under Architecture_ — Book Design

|                  |                                                                                                                                                                                                                                                                        |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Work**         | _File Under Architecture_ by Herbert Muschamp                                                                                                                                                                                                                          |
| **Year**         | 1974                                                                                                                                                                                                                                                                   |

| **Context**      | Printed entirely on brown Kraft paper, typeset on an IBM Selectric typewriter — one of the first books to be typeset directly on a computer by the book designer. Only available typeface was monospaced Courier. Cooper used the limitations as creative constraints. |

## 6. "Messages and Means" Course Poster (Rotation System)

|                  |                                                                                                                                                                                                                 |
| ---------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Work**         | "Messages and Means" poster / Rotation System prints                                                                                                                                                            |
| **Year**         | c. 1974                                                                                                                                                                                                         |
| **Medium**       | Offset printing, multi-pass rotation technique                                                                                                                                                                  |
| **Context**      | Cooper's signature technique for the VLW's foundational course. Square sheets run through offset press 4 times, rotating and changing inks with every pass. Created swirling, kinetic typographic compositions. |
| **Collaborator** | Ron MacNeil (co-teacher); VLW students                                                                                                                                                                          |

![Messages and Means — Rotation System prints](https://eyeondesign.aiga.org/muriel-coopers-visions-of-a-future/)

## 8. Cooper Self-Portraits (Video/Slow-Scan Television)

|                  |                                                                                                                                                                                                                                                                                                  |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Work**         | Self-portraits                                                                                                                                                                                                                                                                                   |
| **Year**         | 1972 (captured) / early 1980s (printed)                                                                                                                                                                                                                                                          |
| **Medium**       | Video still → slow-scan television → print                                                                                                                                                                                                                                                       |
| **Context**      | Cooper captured herself snapping a photo with a Polaroid camera using a black-and-white Sony Portapak video camera in 1972. Roughly a decade later, she broadcast the video stills via slow-scan television (technology used to transmit images from the moon), creating a unique set of prints. |
| **Collaborator** | —                                                                                                                                                                                                                                                                                                |

## 9. VLW Stacked & Distorted Type Experiments (1980s)

|                  |                                                                                                                                                                                                      |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Work**         | Computer-generated stacked and distorted typography                                                                                                                                                  |
| **Year**         | 1980s                                                                                                                                                                                                |
| **Medium**       | Computer graphics / digital typography                                                                                                                                                               |

## 11. Information Landscapes (TED5 Presentation)

|                   |                                                                                                                                                                                                                                                                                                                                                                                                                         |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Work**          | Information Landscapes                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Year**          | 1994                                                                                                                                                                                                                                                                                                                                                                                                                    |
| **Medium**        | Interactive 3D digital typography / data visualization software                                                                                                                                                                                                                                                                                                                                                         |
| **Context**       | Cooper's culminating work, presented at TED5 in Monterey, CA. Dynamic, navigable 3D typographic environments running on a Silicon Graphics Reality Engine ($250,000). Users could fly through "landscapes" of data. Bill Gates requested a copy. Sub-projects: Typography in Space, Financial Viewpoints, GeoSpace, Interactive Calendar, Network Multiviews, Filtering the News, Personalized Galaxies of Information. |
| **Collaborators** | David Small (code), Suguru Ishizaki, Lisa Strausfeld, Ron MacNeil, Henry Lieberman, Louis Weitzman, Robin Kullberg, Ishanta Lokuge, Earl Rennison, Robert Silvers, Jeffrey Ventrella, Yin Yin Wong, Xiaoyang Yang                                                                                                                                                                                                       |

![Information Landscapes — TED5 Conference, 1994 (video thumbnail)](https://i.ytimg.com/vi/BhrZHkdc2rU/maxresdefault.jpg)
