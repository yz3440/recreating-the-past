## Homework

Your homework every week is to recreate one work of the artist / artists we discuss that week using code, any tool you'd like (as long as it's computational).

In the homework document you will show the original and the recreation side by side, and include any notes about the process, screenshots, animation, etc from your recreation. Also please link to the original code you used to make the work.

In addition please highlight / write down one sentence from the assigned reading for reading back in class.
